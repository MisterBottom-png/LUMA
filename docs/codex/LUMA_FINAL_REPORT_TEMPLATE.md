# LUMA Final Report Template

Every coding task must end with this report.

```text
Request type:
Visual Fix / Bug Fix / Small Implementation / Broad Feature Discovery / UX/Product Exploration / Risky Technical Feature / Review Only / Learning Only

What I changed:
- ...

Files changed:
- ...

Validation run:
- command: result
- command: result
- temporary debug build download: url, local server URL, or not applicable
- not run: reason

What I checked visually:
- ...

Related flows checked:
- ...

Risks or limitations:
- ...

Manual test steps:
1. ...
2. ...
3. ...

Learning updates:
- updated: ...
- not needed: reason

Safe to continue:
Yes / No / Yes with notes
```

For review-only tasks, use:

```text
Request type:
Review Only

Must-fix:
- ...

Should-fix:
- ...

Safe-to-ignore:
- ...

Affected files/screens:
- ...

Recommended implementation prompt:
...
```

For broad feature discovery, use:

```text
Request type:
Broad Feature Discovery

Clarifying questions:
1. ...
2. ...

No app code changed.
```
